# GLPyJson2Model
PyQt5 写的Json转Model工具, 支持多平台Mac,Windows,Linux

功能:

* 根据`JSON`字符串生成`Model Class`代码, 目前只生成`Python`语言代码.

## How To Use?

直接运行 dist 目录下的可执行文件:

* Mac: `/dist/GLPyJson2Model.app`
* Windows: `GLPyJson2Model.exe`
